# vue3-base-project

Base Vue3 + Vite + TS + Vue-Router + Pinia template.

## Project Setup

```sh
npm ci
npm run prepare
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with ESLint

```sh
npm run lint
```
